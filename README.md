## 🛠 Setup Instructions

### Option A: Quick Setup (Recommended)
1. **Create Repository**: Name it `your-username.github.io` (replace with your GitHub username)
2. **Upload Files**: Upload all 4 files (`index.html`, `style.css`, `script.js`, `README.md`)
3. **Auto-Deploy**: Your site will be automatically live at `https://your-username.github.io`

### Option B: Custom Repository Name
1. **Create Repository**: Name it `resume` (or any name you prefer)
2. **Upload Files**: Upload all 4 files to the repository
3. **Enable Pages**: 
   - Go to Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main" + "/ (root)"
4. **Access**: Site will be live at `https://your-username.github.io/resume`

### ⚡ Fast Track Setup (5 minutes)
1. Fork or download these files
2. Create new repository on GitHub
3. Drag and drop all files to GitHub
4. Wait 2-3 minutes for deployment
5. Your professional resume is live! 🎉

## 🎨 Customization Options

### Update Your Information
Edit `index.html` to customize:
- Contact information
- Profile image URL
- Achievement numbers
- Work experience details
- Skills and tools
- Education background

### Change Colors/Styling
Edit `style.css` variables at the top:
```css
:root {
    --primary-color: #2c3e50;
    --accent-color: #e74c3c;
    --gradient-primary: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### Add New Sections
Follow the existing HTML structure to add:
- Portfolio gallery
- Testimonials
- Blog posts
- Certifications
